### gemini-agent-langchain

Project tooling:

- [UV](https://docs.astral.sh/uv/guides/projects/#managing-dependencies) for pyhone env and dependencies.

Download dependencies

```bash
uv sync
```

Use `env`

```bash
cp format.env .env
```

Fill all the values in the `.env`

```bash
source .venv/bin/activate
```

```bash
python3 main.py
```
